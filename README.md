# 🎖️ Team Ideaprism (팀 아이디어프리즘)

창균 대표님의 1인 기업 성장을 돕는 AI 전문가 팀의 페르소나와 자산 관리 저장소입니다.

## 👥 팀 구성
- **코다리 개발수석 (`kodari_chief.md`)**: 기술적 조언 및 팀 총괄 (Manager)
- **기획팀장 (`planning_manager.md`)**: 서비스 기획 및 데이터 분석
- **홍보팀장 (`pr_manager.md`)**: 마케팅 전략 및 트렌드 분석
- **대외협력팀장 (`partnership_manager.md`)**: 파트너십 및 투자 유치

> **참고**: 모든 페르소나는 `.md`(마크다운) 버전과 `.txt` 버전을 모두 제공합니다. 안티그래비티 환경에서는 더 풍부한 표현이 가능한 `.md` 사용을 권장합니다.

## 🚀 사용 방법
이 저장소는 집과 회사 등 여러 작업 공간에서 동일한 에이전트 환경을 유지하기 위해 설계되었습니다.

1. **GitHub 연동**: 이 폴더를 GitHub 저장소로 관리하여 환경을 동기화합니다.
2. **에이전트 소환**: Antigravity(AI 어시스턴트)에게 해당 페르소나 파일을 읽도록 요청하세요.
   - 예: "이 폴더의 `kodari_chief.md` 지침에 맞춰서 대답해줘."
3. **이미지 자산 (Raw URL)**: 모든 페르소나 파일 내의 이미지는 본 저장소의 `raw.githubusercontent.com` 주소를 사용하여 외부 환경에서도 끊김 없는 렌더링을 보장합니다.

## 📂 폴더 구조
- `images/`: 각 팀원의 표정 이미지가 저장된 폴더 (Raw 이미지 소스)
- `*.md`: 시각적 표현이 강화된 에이전트 페르소나 설정 (권장)
- `*.txt`: 호환성을 위한 텍스트 버전 페르소나 설정
- `.agent/workflows/`: Antigravity용 자동화 워크플로우 (선택 사항)
